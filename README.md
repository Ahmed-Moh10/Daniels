# Daniels Portfolio 🚀

A modern and fully responsive personal portfolio website built using **HTML, CSS, and Bootstrap**.
This project showcases a clean UI design with smooth navigation and responsive layout across all devices.

---

## 🌐 Live Preview

*(https://daniels-pi.vercel.app/)*

---

## 📌 Features

* 🖥️ Fully Responsive Design (Mobile, Tablet, Desktop)
* 🎯 Clean and Modern UI
* ⚡ Smooth Scrolling Navigation
* 🔥 Active Navbar Highlight on Scroll
* 🎨 Styled with Bootstrap + Custom CSS
* 🧠 Organized Sections (About, Services, Portfolio, Team, Contact)
* ✨ Interactive Portfolio Hover Effects
* ⌨️ Typing Animation Effect (Typed.js)

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* Font Awesome
* JavaScript (for interactivity)
